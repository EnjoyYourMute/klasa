Class: Schema

Schema

The schema class that stores (nested) folders and keys for SettingGateway usage. This class also implements multiple helpers.

new Schema(client, gateway, options, parent, key)

Parameters:
Name Type Argument Description
client KlasaClient The client which initialized this instance
gateway Gateway The Gateway that manages this schema instance
options Object The object containing the properties for this schema instance
parent Schema <nullable>
 The parent which holds this instance
key string The name of this key
