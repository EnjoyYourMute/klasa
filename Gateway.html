Class: Gateway

Gateway

The Gateway class that manages the data input, parsing, and output, of an entire database, while keeping a cache system sync with the changes.

new Gateway(store, type, validateFunction, schema, options)

Parameters:
Name Type Description
store GatewayDriver The GatewayDriver instance which initiated this instance
type string The name of this Gateway
validateFunction function The function that validates the entries' values
schema Object The initial schema for this instance
options GatewayOptions The options for this schema
